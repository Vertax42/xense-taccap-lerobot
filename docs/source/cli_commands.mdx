# CLI Command Reference

Quick reference for `lerobot-teleoperate` and `lerobot-record` commands
for all Xense-supported robots.

---

## lerobot-teleoperate

### ARX5 follower (teach mode)

```bash
lerobot-teleoperate \
    --robot.type=arx5_follower \
    --robot.arm_port=can1 \
    --robot.control_mode=joint_control \
    --robot.id=left \
    --teleop.type=mock_teleop \
    --fps=30 \
    --display_data=true
```

### ARX5 follower + trlc_leader (joint master-slave teleop)

```bash
lerobot-teleoperate \
    --robot.type=arx5_follower \
    --robot.arm_port=can1 \
    --robot.control_mode=joint_control \
    --robot.id=left \
    --teleop.type=trlc_leader \
    --teleop.port=/dev/ttyACM0 \
    --fps=30 \
    --display_data=true
```

### ARX5 follower + SpaceMouse (default cartesian control_mode)

```bash
lerobot-teleoperate \
    --robot.type=arx5_follower \
    --robot.arm_port=can1 \
    --robot.id=left \
    --teleop.type=spacemouse \
    --teleop.id=left \
    --fps=100
```

### ARX5 bimanual (teach mode)

```bash
lerobot-teleoperate \
    --robot.type=bi_arx5 \
    --robot.id=bimanual \
    --teleop.type=mock_teleop \
    --fps=30 \
    --debug_timing=false \
    --display_data=true
```

## lerobot-record

> Replace `<my_username>/<my_dataset_name>` with your actual HuggingFace repo path.

### Generic robots (SO-100, SO-101, Koch, etc.)

```bash
lerobot-record \
    --robot.type=so101_follower \
    --robot.port=/dev/tty.usbmodem58760431541 \
    --robot.id=black \
    --teleop.type=so101_leader \
    --teleop.port=/dev/tty.usbmodem58760431551 \
    --teleop.id=blue \
    --dataset.repo_id=<my_username>/<my_dataset_name> \
    --dataset.num_episodes=50 \
    --dataset.single_task="Pick up the cube" \
    --dataset.episode_time_s=60 \
    --dataset.reset_time_s=30 \
    --dataset.streaming_encoding=true \
    --dataset.encoder_threads=2 \
    --display_data=true
```
