# CLI Command Reference

Quick reference for `lerobot-teleoperate` and `lerobot-record` commands
for all Xense-supported robots.

---

## lerobot-teleoperate

### Generic robots (SO-101, Koch, etc.)

```bash
lerobot-teleoperate \
    --robot.type=so101_follower \
    --robot.port=/dev/tty.usbmodem58760431541 \
    --robot.cameras="{ front: {type: opencv, index_or_path: 0, width: 1920, height: 1080, fps: 30}}" \
    --robot.id=black \
    --teleop.type=so101_leader \
    --teleop.port=/dev/tty.usbmodem58760431551 \
    --teleop.id=blue \
    --display_data=true
```

### ARX5 single-arm (teach mode)

```bash
lerobot-teleoperate \
    --robot.type=arx5_follower \
    --robot.arm_port=can0 \
    --robot.id=right \
    --teleop.type=btgamepad \
    --teleop.id=right \
    --fps=30 \
    --display_data=true
```

### ARX5 single-arm + SpaceMouse

```bash
lerobot-teleoperate \
    --robot.type=arx5_follower \
    --robot.arm_port=can0 \
    --robot.id=right \
    --teleop.type=spacemouse \
    --teleop.id=right \
    --fps=30
```

### ARX5 bimanual (teach mode)

```bash
lerobot-teleoperate \
    --robot.type=bi_arx5 \
    --robot.left_config.arm_port=can0 \
    --robot.right_config.arm_port=can1 \
    --robot.id=bimanual \
    --teleop.type=btgamepad \
    --teleop.id=bimanual \
    --fps=30
```

## lerobot-record

> Replace `<my_username>/<my_dataset_name>` with your actual HuggingFace repo path.

### Generic robots (SO-100, SO-101, Koch, etc.)

```bash
lerobot-record \
    --robot.type=so101_follower \
    --robot.port=/dev/tty.usbmodem58760431541 \
    --robot.id=black \
    --teleop.type=so101_leader \
    --teleop.port=/dev/tty.usbmodem58760431551 \
    --teleop.id=blue \
    --dataset.repo_id=<my_username>/<my_dataset_name> \
    --dataset.num_episodes=50 \
    --dataset.single_task="Pick up the cube" \
    --dataset.episode_time_s=60 \
    --dataset.reset_time_s=30 \
    --dataset.streaming_encoding=true \
    --dataset.encoder_threads=2 \
    --display_data=true
```
